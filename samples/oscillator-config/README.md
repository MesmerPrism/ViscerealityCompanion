# Oscillator Config Samples

These sample profiles mirror the public configuration surface from the
Viscereality oscillator asset while keeping the actual runtime dynamics out of
this repo.

- `profiles.json` is the catalog manifest the Windows app loads.
- `studio-baseline.json` is a balanced default operator preset.
- `twin-coherence.json` biases the public config toward coherence-driven twin
  sessions.

The private overlay may consume these documents directly or translate them into
local-only runtime assets.
